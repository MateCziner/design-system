# Typography

Design System have defined two basic font families. As base font family we are using sans-serif
`ProximaNova` font, for monospaced text we are using `SpaceMono` font.

## Font licensing

### ProximaNova

ProximaNova is licensed font that cannot be bundled and redistributed along with this package. If you
want to use this font you have to obtain your own license for this font. You can use for example
[Adobe Fonts](https://fonts.adobe.com/fonts/proxima-nova) service.

### SpaceMono

SpaceMono is licensed under the [Open Font License](https://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&id=OFL)
and is bundled within this package.
